# 🌐 Personal Portfolio Website

A modern, responsive, and interactive full-stack developer portfolio built to showcase my skills, projects, experience, and resume. This portfolio serves as my digital resume, highlighting real-world projects, technical expertise, and providing an easy way for recruiters and collaborators to connect with me.

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern and clean UI with smooth animations
- Professional hero section
- Skills & technology stack
- Featured projects with GitHub and live demo links
- About & career timeline
- Downloadable resume
- Functional contact form
- SEO-friendly structure

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js (Contact Form)
- Git & GitHub

- 🥗 NutriScan
- 🎓 StudyNotion
- 📊 Customer Shopping Dashboard
